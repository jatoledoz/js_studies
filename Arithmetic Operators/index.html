Arithmetic Operators

  • Addition or concatenate (+) - it returns a addition if values are numbers and concatenate if values are strings
  • Subtraction (-) - it returns difference between values
  • Multiplication (*) - it returns the product of values
  • Division (/) - it returns the quotient of values
  • Modulus (%) - it returns the remainder after the division
  • Increment (++) - pre/post increment
  • Decrement (--) - pre/post decrement
