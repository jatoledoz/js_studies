Operadores:

(==) - checks if values compared are equal

(===) - checks if values compared are equal and the same type

(!=) - checks if values compared are different

(!==) - checks if values compared are different and different types

(<) - checks if value on left is less than the one on right

(>) - checks if value on left is more than the one on right

(<=) - checks if value on left is less or equal than the one on right

(>=) - checks if value on left is more or equal than the one on right
